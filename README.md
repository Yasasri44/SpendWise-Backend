# 💳 SpendWise Backend

Spring Boot REST API for Expense Tracking Application

## 🚀 Features
- Add Expense
- Delete Expense
- Fetch All Expenses

## 🛠 Tech Stack
- Java
- Spring Boot
- MongoDB

## 📡 API Endpoints

GET /expenses  
POST /expenses  
DELETE /expenses/{id}

## ▶️ Run Locally
1. Start MongoDB
2. Run Spring Boot Application
3. Runs on http://localhost:8080
